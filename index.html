Project Overview

FitFlow is a full-stack fitness tracking platform that helps users monitor their nutrition, workouts, progress photos, and fitness goals with advanced analytics and an AI-powered coach. Built with modern web technologies and containerized with Docker for seamless deployment.

  • Dashboard Overview: Real-time fitness summary with key metrics including weekly workouts, daily calories, current weight & BMI, and active streak

  • Nutrition Tracker: Track daily meals by type (Breakfast, Lunch, Dinner, Snack) with complete macronutrient breakdown (protein, carbs, fats) and visual charts
  • Progress Gallery: Upload and visualize transformation photos over time to track your fitness journey
  • Goals Management: Set measurable fitness milestones with visual progress bars and celebrate achievements
  • Analytics & Stats: 4 interactive charts showing workout frequency, calorie intake trends, macros breakdown, and duration patterns
  • BMR/TDEE Calculator: Calculate daily calorie needs based on personal metrics with calculation history
  • Nutrition Trivia: Interactive quiz game to test and improve your nutrition knowledge
  • AI Fitness Coach: Get personalized fitness and nutrition advice 24/7 powered by Mistral AI
  • Profile Management: Complete user profile with the ability to update personal info, fitness goals, and change password securely

  • DevOps: Docker & Docker Compose, Nginx reverse proxy, Multi-stage builds for optimization
  • Testing: Jest & Supertest with 90+ comprehensive tests covering all API endpoints

  • Docker Containerization: Complete Docker Compose setup with health checks, volume persistence, and optimized multi-stage builds
  • Security: JWT-based authentication, bcrypt password hashing, secure password requirements (8+ chars, uppercase, lowercase, number, special char)

  • Change Data Capture: TiCDC streaming database changes to Kafka in real-time

  • API Documentation: Interactive Swagger/OpenAPI docs at /api-docs endpoint
  • Docker Deployment: Complete orchestration with docker-compose (one-command startup)

Tech Stack

  • Backend: Node.js, Express, JWT authentication, bcrypt password hashing
  • Database: TiDB (distributed SQL), TiKV (storage engine), PD (placement driver)
  • CDC & Messaging: TiCDC for change data capture, Kafka with Zookeeper for event streaming
  • API Documentation: Swagger/OpenAPI for interactive API documentation
  • DevOps: Docker & Docker Compose with multi-service orchestration
  • Logging: log4js for structured application logs
  • Security: CORS, rate limiting, input validation, SQL injection protection

Technical Achievements

  • Event-Driven Architecture: Built microservices with Kafka for asynchronous event processing
  • Database Streaming: Integrated TiCDC to capture and stream all database changes to Kafka topics
  • One-Command Deployment: Complete Docker Compose setup initializing 8+ services automatically
  • API Documentation: Comprehensive Swagger documentation with interactive testing interface
  • Token Management: Implemented refresh token rotation for secure long-lived sessions
  • Rate Limiting: Protected authentication endpoints with intelligent rate limiting (100 req/15min general, 10 req/min auth)
  • Error Handling: Comprehensive error handling with meaningful error messages and proper HTTP status codes
  • Production Ready: Complete with health checks, logging, monitoring, and security best practices

API Endpoints

  • POST /api/auth/register: Create new user account with validation
  • POST /api/auth/login: Authenticate and receive access + refresh tokens
  • POST /api/auth/refresh: Get new access token using refresh token
  • POST /api/auth/logout: Invalidate current session
  • GET /api/auth/validate: Verify token validity
  • GET /api/users: List all users (protected route)
  • GET /api/users/:id: Get specific user details (protected route)
  • GET /health: Application health check endpoint
  • GET /api-docs: Interactive Swagger API documentation
